The Goat language

A small functional programming language with a tree-walking interpreter, an interactive REPL, and a real standard library. Written in F#.

fibonacci.goat 
fun rec fib n =
  match n with
  | 0 => 0
  | 1 => 1
  | n => fib (n - 1) + fib (n - 2)

let fibStream = fibs 0 1

fun main _ =
  io {
    do! println (show (take 10 fibStream))
  }

What the language gives you

Functional core

First-class functions, closures, lambdas, partial application, currying, and tail recursion.

Lazy evaluation

Explicit laziness with lazy and force. Infinite lists, streams, and corecursion.

Algebraic data types

Multi-constructor ADTs, nested data, and exhaustive pattern matching with guards.

IO monad

io { } blocks with do! and let! for clean, sequenced input and output.

Standard library

Option, Result, list combinators, string helpers, and math, written in Goat itself.

REPL and CLI

An interactive shell, the run and check commands, and a Makefile for quick iteration.

Quick start

1

Clone the repository

git clone https://github.com/GoatWhistle/goat-lang
cd goat-lang/compiler
2

Build the project

dotnet restore Goat.slnx
dotnet build Goat.slnx
3

Run the REPL or a program

dotnet run --project src/Goat.Repl
dotnet run --project src/Goat.Cli -- run examples/factorial.goat
4

Or use the Makefile

make repl
make run FILE=examples/fibonacci.goat
make test
