Operators

Operators are listed from lowest to highest precedence.

PrecedenceOperatorAssociativityDescription
1|>leftPipe: x |> f becomes f x
1~>leftLazy pipe: the argument is wrapped in lazy
2&&leftLogical AND
2||leftLogical OR
3== != /=noneEquality / inequality (!= and /= are synonyms)
3< > <= >=noneComparison
4++rightString and list concatenation
5::rightCons: prepend an element to a list
6+ -leftAddition / subtraction
7* / modleftMultiplication / division / remainder
8**rightExponentiation
9notprefixLogical negation
10-prefixUnary minus
11applicationleftf x y, function application
 
-- Operator examples
let a = 2 ** 10              -- 1024
let b = 10 mod 3            -- 1
let s = "hello" ++ " world" -- "hello world"
let xs = 1 :: 2 :: 3 :: []  -- [1, 2, 3]
let r = xs |> reverse |> show  -- pipeline

Pipe operators

Feed the value on the left into the function on the right:

-- |> is the ordinary pipe
let result = [1..10]
  |> filter (\x -> x mod 2 == 0)
  |> map (\x -> x * x)
  |> fold (\acc x -> acc + x) 0
  |> show
