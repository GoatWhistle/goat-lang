Combinators & utilities

Small functional building blocks.

FunctionTypeDescription
id xa→aThe identity function
const x _a→b→aAlways returns the first argument
flip f x y(a→b→c)→b→a→cSwaps the argument order
compose f g x(b→c)→(a→b)→a→cFunction composition: f(g(x))
fix f((a→b)→a→b)→a→bFixed point (the Y combinator)
 
let double    = \x -> x * 2
let addOne    = \x -> x + 1
let doubleThenAdd = compose addOne double
let r = doubleThenAdd 5  -- 11

let always42 = const 42
let xs = map always42 [1..5]  -- [42, 42, 42, 42, 42]
