Installation & CLI

Build Goat from source and run programs through the CLI or the interactive REPL.

Requirements

ToolVersion
.NET SDK10.0.x
F#9+
OSWindows / Linux / macOS

Build from source

git clone https://github.com/GoatWhistle/goat-lang
cd goat-lang/compiler
dotnet restore Goat.slnx
dotnet build Goat.slnx
dotnet test Goat.slnx

CLI commands

CommandAction
goat run file.goatParse, evaluate, and run main
goat check file.goatParse and desugar without running
goat replStart the interactive REPL
goat --helpShow usage
goat --versionShow the interpreter version
 
dotnet run --project src/Goat.Cli -- run examples/factorial.goat
dotnet run --project src/Goat.Repl

Makefile targets

CommandAction
make buildCompile
make test142 xUnit tests
make replStart the REPL
make run FILE=…Run a file
make examplesRun every example
